


*****LATEST COMMANDS FOR NEW PROJECT******
1. create hugo project by hugo new site quickstart
2. add theme="narrow"
3. add important data from hugo.toml (previously config.toml)
4. add authors in data folder + add files in content + take care of static folder as well then follow the steps for exisiting projects


*****LATEST COMMANDS FOR EXISTING PROJECT******
1. hugo server OR hugo server -D --bind 192.168.29.223 --baseUrl http://192.168.29.223 [use ifconfig command IP:1313]
2. hugo new posts/news/category/post-name.md
3. check vs code for article 
4. edit article 
5. check article on chrome http://localhost:1313/
6. publish article on github desktop.
